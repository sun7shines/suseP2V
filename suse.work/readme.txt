

把物理主机p2v 至虚拟机

1 根据物理机的suse版本，使用对应的iso安装虚拟机

2 把 目录suse.work拷贝至虚拟机环境中。

3 执行 python find_modules.py ，获取当前虚拟机的所有相关硬件驱动。会在当前目录生成一个内核版本号命名的文件夹，里面包含相关的驱动

以上步骤在虚拟机内部执行

========================================================================================

把上述虚拟机内部的suse.work 目录 拷贝至物理主机。

4 python  update_phy_os.py 。

5 关闭物理主机，执行p2v操作

6 把系统文件复原。使用p2v光盘引导系统（如果不能进行引导，无法识别主机的文件系统如reiserfs，则需要用物理主机原安装光盘进行引导），挂载物理主机的文件系统,重新覆盖原文件备份目录位于/var/syssbackup目录中
